Introdução ao JavaScript

JavaScript é uma linguagem de programação de alto nível que todos os navegadores modernos suportam. É também uma das principais tecnologias da web, juntamente com HTML e CSS que você pode ter aprendido anteriormente. Esta seção abordará os conceitos básicos de programação JavaScript, que variam de variáveis ​​ e aritmética a objetos e loops.
